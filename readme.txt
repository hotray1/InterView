1. 赋值运算符：
简单：var userName="tom";
复合赋值运算符：
a+=b;  // 相当于 a=a+b;
a-=b;  // 相当于 a=a-b;
a*=b;  // 相当于 a=a*b;
a/=b;  // 相当于 a=a/b;
a%=b;  // 相当于 a=a%b;
a&b=b; // 相当于 a=a&b;
a|=b;  // 相当于 a=a|b;
a^=b;  // 相当于 a=a^b;

2. 算术运算符
 ++ i=1;j=i++; // j是1,i是2.因为i++是先用后加
    i=1;j=++i  // j是2,i是2.因为++i是先加后用
    除法运算时，0不能作为除数，如0是除数，则返回结果为 Infinity。
    
3. 比较运算符，返回布尔值
== 等于，只根据表面值进行判断，不涉及数据类型
=== 全等于 不仅判断表面值，还要判断数据类型是否一样
